# VibeCoding 勉強会

AIに話しかけるだけでアプリが作れる「VibeCoding」を体験する勉強会の資材リポジトリです。

---

## クイックスタート

### 1. リポジトリを Clone する

```bash
git clone https://github.com/YOUR_USERNAME/vibecoding-tutorial.git
```

> GitHub アカウントと Git のインストールが必要です。詳しくは [`docs/02-setup-codex.md`](docs/02-setup-codex.md) を参照してください。

### 2. アプリをブラウザで開く

Clone したフォルダの中の `starter-app/index.html` をダブルクリックして、ブラウザで開きます。

### 3. Codex App でフォルダを開く

OpenAI Codex App を起動し、Clone したフォルダ（`vibecoding-tutorial`）を開きます。  
あとは日本語で話しかけるだけです！

## 勉強会当日の流れ

1. `docs/01-what-is-vibecoding.md` を読んで VibeCoding の概要を理解する
2. `docs/02-setup-codex.md` の手順で環境を整える
3. `starter-app/index.html` をブラウザで開いてアプリを確認する
4. Codex App を使ってアプリに機能を追加していく
5. 作ったものをみんなで共有する 🎉
