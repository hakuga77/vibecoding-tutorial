# 出張プランナー（スターターアプリ）

## このアプリについて

行き先・出発日・帰着日・目的を入力すると、出張の持ち物チェックリストを表示するシンプルなアプリです。

- **ファイル**: `index.html`（ブラウザで直接開けます）
- **技術**: HTML / CSS / JavaScript のみ（インストール不要）

---

## 起動方法

`index.html` をダブルクリックするだけで、ブラウザで開けます。

---

## Codex で改造してみよう！

このアプリはわざとシンプルに作ってあります。  
Codex に話しかけて、あなただけのアプリに育てていきましょう。

### 簡単な改造アイデア

| 難易度 | アイデア |
|--------|---------|
| ★☆☆ | ボタンの色を変える |
| ★☆☆ | 背景色を変える |
| ★★☆ | 出発までのカウントダウンを表示する |
| ★★☆ | 泊数を自動計算して表示する |
| ★★☆ | 持ち物リストをカテゴリ別に整理する |
| ★★★ | 予算欄と1日あたり費用の計算を追加する |
| ★★★ | 印刷ボタンを追加する |

### 指示の例

```
出発日から帰着日まで何泊か自動で計算して「〇泊〇日」と表示してください
```

```
持ち物リストを「仕事道具」「衣類」「日用品」の3つのカテゴリに分けてください
```

詳しくは [`../docs/04-prompt-tips.md`](../docs/04-prompt-tips.md) を参照してください。
