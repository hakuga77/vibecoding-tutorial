# VibeCoding とは？

## ひと言で言うと

> **「コードを書かずに、AIに話しかけてアプリを作る開発スタイル」**

プログラミングの知識がなくても、「こんなものを作って」と日本語で伝えるだけで、AI がコードを書いてくれます。

---

## 従来の開発との違い

### 従来の開発
```
アイデア → 設計 → エンジニアがコードを書く → テスト → 完成
（数日〜数週間）
```

### VibeCoding
```
アイデア → AIに話しかける → 数分で動くものができる
```

エンジニアとの連携を待つ必要がなく、アイデアをその場で形にできます。

---

## 何が変わるの？

| 従来 | VibeCoding |
|------|-----------|
| コードを書く必要がある | 日本語で話しかけるだけ |
| エンジニアに依頼が必要 | 自分で作れる |
| 数日〜数週間かかる | 数分〜数時間で完成 |
| 専門知識が必要 | 誰でもできる |

---

## こんなことができます

### 業務ツールをサクッと作る
- 「出張の持ち物チェックリストを作るアプリを作って」
- 「議事録のテンプレートをすぐに出力できるページを作って」
- 「毎週の報告書フォーマットを入力できるツールを作って」

### 見た目を変える
- 「もっとかっこいいデザインにして」
- 「スマホでも使いやすくして」
- 「会社のブランドカラーに合わせて」

### 機能を追加する
- 「印刷ボタンを追加して」
- 「入力した内容をメモとして保存できるようにして」
- 「グラフで表示して」

---

## VibeCoding で大事なこと

### 1. 具体的に伝える
- ❌ 「いい感じにして」
- ✅ 「背景を白にして、タイトルの文字を大きく太くして」

### 2. 一度に一つのことをお願いする
- ❌ 「デザインを直して機能も追加して」
- ✅ まず「デザインを直して」→ 確認 → 次に「機能を追加して」

### 3. うまくいかなくても繰り返せる
- AIは何度でも修正してくれます
- 「やっぱり元に戻して」「もっとシンプルにして」も OK

---

## 今日のハンズオンで体験すること

「出張プランナー」という簡単なアプリをベースに、みなさん自身が Codex に指示を出して、アプリを育てていきます。

コードを一切書かなくても、あなたのアイデア次第でアプリはどんどん進化します。
