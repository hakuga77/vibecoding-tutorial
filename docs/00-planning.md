# VibeCoding 勉強会 企画書

## 概要

| 項目 | 内容 |
|------|------|
| 目的 | 社内メンバーが VibeCoding（AIとの対話でアプリを作る開発スタイル）を体験し、業務活用のイメージを掴む |
| 対象 | 非エンジニア中心（コードを書いたことがない方でも参加可） |
| 使用ツール | OpenAI Codex（Windows ネイティブアプリ） |
| 形式 | ハンズオン（手を動かして体験） |
| 想定時間 | 1.5〜2時間 |

---

## 勉強会で伝えたいこと

1. **VibeCoding とは何か**
   - コードを書かずに、AIに話しかけるだけでアプリが作れる
   - エンジニアと非エンジニアの壁が溶ける新しい開発スタイル

2. **Codex を使えば誰でもできる**
   - インストールして、日本語で指示するだけ
   - プログラミング知識ゼロでも大丈夫

3. **業務に使えるイメージを持って帰ってもらう**
   - 「これなら自分でも作れそう」という体験を提供する

## ハンズオンで作るアプリ：出張プランナー

参加者は「出張プランナー」アプリをベースに、Codex を使って機能追加・デザイン改善を体験する。

### 初期状態（最小限の機能）
- 行き先・出発日・帰着日・目的を入力するフォーム
- 「プランを作成」ボタンを押すと、基本的な持ち物チェックリストが表示される

### 参加者が挑戦する拡張例
- 「持ち物リストをカテゴリ別に整理して」
- 「出発日までのカウントダウンを表示して」
- 「デザインをビジネスらしくかっこよくして」
- 「予算メモ欄を追加して」
- 「印刷ボタンを追加して」
